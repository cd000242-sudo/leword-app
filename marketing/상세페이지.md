## 한 줄 소개

LEWORD는 네이버 블로그 운영자를 위한 키워드 발굴 + 노출 추적 도구입니다.
시맨틱 sibling 확장, 황금비율 자동 필터, 내 글 SERP 추적, 26 카테고리 헌터까지 — 모두 한 화면에서.

## 핵심 기능 (v2.42.91)

### 💎 황금 키워드 자동 발굴 (네이버 연관 키워드의 상위호환)

- 입력 키워드 하나로 시맨틱 sibling 자동 확장 — 헤드 명사 공유, modifier 다른 의외의 키워드 발굴
- 네이버 SearchAd API 로 모든 후보의 검색량 + 문서수 실측
- 황금비율 자동 필터: 검색량 ≥300, 문서수 ≤10000, 비율 ≥5
- 결과는 비율 ↓ 정렬 → 즉시 글쓰기 후보

**기존 키워드 조회와의 차이**
- 키워드 조회: 자동완성/연관검색 위주 → prefix 확장만 (예: "황금키워드 찾는법")
- 황금 발굴: 시맨틱 sibling 추가 → 의외의 동의어/유사어 (예: "꿀키워드", "롱테일 키워드 발굴")

### 🏠 홈판 키워드 헌터 (26 카테고리)

- beauty / fashion / food / parenting / pet / health / finance / career / realestate / travel / camping / hobby / wedding / car / entertainment / music / book / game / sports / it / education / celebrity / issue 등 26개 카테고리
- 각 카테고리 시드 → 자동완성 다단계 확장 → 검색량/문서수/SERP 빈자리/신선도 4종 실측
- homeScore 산출 + summary (예: "🛑 빅도메인 독점 — 신생 진입 불가")
- 추천 제목 5종 자동 생성 + 패턴 매칭

### 📊 내 노출 추적 (검증 루프)

- 네이버 블로그 ID/주소 한 번 입력 → 자동 RSS 변환 + 접근성 검증
- RSS 글 자동 수집 → 글 제목에서 핵심 키워드 자동 추출 → 페어 자동 등록
- 네이버 블로그 검색 API 우선 사용 (차단 없음), HTTP SERP + Playwright 폴백
- 카테고리별 hit rate 자동 집계 — 2~3주 누적 시 본인 강점/약점 카테고리 자동 식별
- 자동 파이프라인: 모달 열면 fetch → match → SERP 체크 자동 (5분 쓰로틀)
- 특정 글 + 키워드 페어 수동 등록 가능

### 🆕 연예 갓 떴음 (4매체 통합)

- 스타뉴스 + 스포츠조선 + 디스패치 + 마이데일리 동시 크롤
- 최근 3시간 이내 + HOT 미진입 = 선점 가능 기사만 자동 필터
- 분 단위 시간 + 매체/카테고리 배지 + 🟢선점/⚠️경쟁 표시
- 5분마다 자동 갱신

### 🔗 마인드맵 (시맨틱 확장)

- 자동완성 + 연관검색 + 시맨틱 sibling + 롱테일 4중 확장
- 50~500개 후보 다단계 발굴
- 등급 자동 분류 (SSS/SS/S/A/B)
- 엑셀 한 번에 내보내기

### 📎 신호 근거 보기 (신뢰성)

- 모든 홈판 카드 우하단 [📎 근거 보기] 버튼
- 측정값별 출처 분류:
  - [공식] = C-Rank, D.I.A., D.I.A.+, AiRS 등 네이버 발표
  - [업계 표준] = 블랙키위, 키워드인사이트, 황금키워드마스터 등 다중 도구 합의
  - [휴리스틱] = LEWORD 자체 (공식·표준 합의 없는 신호)
- 각 출처 URL + 핵심 인용 표시

### ⬇️ 트레이 최소화

- 우상단 [⬇️ 트레이로] 버튼 한 번에 트레이로 (결과 보존)
- [X = 트레이] 체크 시 X 버튼도 트레이행
- 트레이 아이콘은 LEWORD 로고 그대로

### 📊 실시간 검색어 5매체

- 네이버 (Signal.bz) + ZUM + 네이트 + 다음 + 연예 갓 떴음 통합
- 5분마다 자동 새로고침
- 클릭 → 외부 브라우저로 기사 열기

## 솔직한 한계 (LEWORD가 약속하지 않는 것)

- **메인 홈피드 진입 100% 보장은 불가**
  - 네이버 홈피드는 AiRS 개인화 추천 알고리즘 — 사용자별 임베딩으로 결정
  - 어떤 도구도 보장 불가, LEWORD도 마찬가지
  - 우리가 할 수 있는 것: "검증된 신호 동시 충족 시 통계적 노출 우위"

- **모든 시맨틱 확장은 룰 기반**
  - LEWORD 는 LLM/임베딩 기반 의미 분석을 사용하지 않습니다 (비용/안정성)
  - 한국어 형태소 분석 없이 휴리스틱(헤드 명사 + 자동완성 다단계 + 토큰 매칭) 사용
  - 그래서 빠르고 안정적이지만, 진짜 동의어("민생지원금" ↔ "긴급생계지원금" 같은) 매칭은 한계

- **노출 추적은 검색 SERP 기준**
  - 메인 홈피드 추적 불가 (위 사유)
  - 검색 SERP 노출 자체가 "잘 만든 글" 신호로 활용됩니다
  - 신생/저권위 블로그는 격전지 키워드에서 top 30 진입 어려움 — 진짜 결과 그대로 보여드림

## 자주 묻는 질문

**Q. 다른 키워드 도구와 무엇이 다른가요?**
A. 블랙키위/키워드인사이트는 키워드 발굴까지. LEWORD는 발굴 + 시맨틱 sibling 확장 +
   내 블로그 노출 자동 추적 + 26 카테고리 헌터까지 통합 제공. 모든 수치 실측 + 출처 명시.

**Q. AI/LLM API 키 필요한가요?**
A. 아니요. 핵심 기능은 모두 네이버 공식 API + SERP 크롤 기반 — 추가 비용 없음.

**Q. 한 PC에서만 쓰나요?**
A. 라이선스 1개당 1 PC. PC 변경 시 라이선스 재발급 가능.

**Q. 환불 정책은?**
A. 결제 후 7일 이내 사용 미발생 시 100% 환불. 상세 약관 확인.
