## 🎯 한 줄 요약

LEWORD는 네이버 블로그 운영자를 위한 **키워드 발굴 + 노출 자동 추적** 통합 도구입니다.
시맨틱 sibling 확장, 황금비율 자동 필터, 내 글 SERP 추적, 26개 카테고리 헌터까지 한 화면에서.

---

## 💡 왜 LEWORD가 필요한가요?

### 문제

- 네이버 연관 키워드 기능이 **공식 종료**되었습니다
- 다른 도구들(블랙키위/키워드인사이트 등)은 **단순 자동완성/연관검색만** 제공 — prefix 확장 위주
- 내 글이 발행 후 **실제 어디서 노출되는지** 추적할 도구가 없음
- 메인 홈피드 노출 약속하는 도구들이 많지만 — **개인화 추천 알고리즘은 본질적으로 블랙박스**

### LEWORD의 답

1. **시맨틱 sibling 확장** — 헤드 명사 공유, modifier가 다른 의외의 키워드 발굴
2. **모든 수치 네이버 API 실측** — 검색량, 문서수, SERP 빈자리, 신선도까지
3. **자동 노출 추적** — RSS로 내 글 수집 → 키워드 매칭 → SERP top 30 자동 측정
4. **모든 신호에 출처 명시** — 공식(C-Rank/D.I.A.+) / 업계 표준 / LEWORD 자체 휴리스틱 3단계 분류
5. **솔직한 한계 인정** — "100% 보장" 약속하지 않습니다

## 🚀 7가지 핵심 기능 (v2.42.91)

### 1. 💎 황금 키워드 자동 발굴 (네이버 연관 키워드의 상위호환)

키워드 하나 입력 → 시맨틱 sibling 확장 → 검색량/문서수 실측 → 황금비율 자동 필터.

**시맨틱 sibling이 뭔가요?**
- 헤드 명사를 공유하지만 modifier가 다른 키워드
- 예: "황금키워드" → "꿀키워드", "롱테일 키워드 발굴", "타겟 키워드"
- 예: "2026 소상공인 지원금" → "고유가 피해지원금", "청년창업지원금", "긴급생계지원금"
- 의외의 동의어/유사어를 자동 발굴

**자동 필터 기준 (실측 기반)**
- 검색량 ≥ 300 (의미 있는 트래픽)
- 문서수 ≤ 10,000 (경쟁 약함)
- 황금비율 ≥ 5 (검색 대비 문서 적음)

기존 "키워드 조회"가 단순 prefix 확장이라면, 황금 발굴은 **의외성 + 가치**를 동시에 잡습니다.

---

### 2. 🏠 홈판 키워드 헌터 (26 카테고리)

beauty · fashion · food · recipe · living · interior · parenting · pet · health · garden ·
finance · career · realestate · travel · camping · hobby · wedding · car · entertainment ·
music · book · game · sports · it · education · celebrity · issue

각 카테고리별 시드 → 자동완성 다단계 확장 → 4개 실측 신호로 homeScore 산출.

**측정 신호 (모두 실측)**
- CTR 가능성 (제목 패턴 매칭)
- 신선도 (Datalab 1h vs 24h surge ratio + 24h 발행수)
- 카테고리 적합도 (C-Rank 공식 신호)
- SERP 빈자리 (인플루언서/빅도메인 vs 일반 블로거 비율)

**보너스**: 추천 제목 5종 자동 생성 (숫자형/호기심갭/시간절박감/결과약속/실경험 패턴)

---

### 3. 📊 내 노출 추적 (검증 루프 — 다른 도구가 못 하는 기능)

내 블로그 ID 한 번 입력 → 자동으로 모든 단계 진행.

**자동 파이프라인**
1. RSS URL 자동 변환 + 접근성 검증
2. 글 50건 자동 수집
3. 글 제목에서 핵심 키워드 자동 추출 (3개/글)
4. 페어 자동 등록 (약 150건)
5. SERP top 30 자동 측정 (네이버 검색 API 우선, 차단 시 Playwright 폴백)
6. 카테고리별 hit rate 자동 집계

**부가 기능**
- 자동 매칭 못 잡은 글은 [➕ 특정 글 노출 확인] 으로 수동 페어 등록
- 블로그 변경 시 [🗑️ 모든 데이터 초기화] 한 번에 정리
- 모달 열 때 5분 쓰로틀로 자동 재측정

**가치**
- 2~3주 누적되면 본인 블로그의 강점/약점 카테고리 자동 식별
- "발굴 → 발행 → 검증" 루프 완성

---

### 4. 🆕 연예 갓 떴음 (4매체 통합 — 선점 가능 기사)

스타뉴스 + 스포츠조선 + 디스패치 + 마이데일리 4개 매체 동시 크롤.

**필터링**
- 최근 3시간 이내 발행
- HOT 미진입 (이미 경쟁 포화된 기사 제외) = **선점 가능 기사만**

**표시 정보**
- 분 단위 시간 (60분 이내는 빨간색 강조)
- 매체별 색상 배지
- 카테고리 자동 분류 (스타/드라마/예능/방송/음악/스포츠)
- 🟢 선점 / ⚠️ 경쟁 배지

5분마다 자동 갱신. 연예/방송 글로 트래픽 잡는 분께 즉시 가치.

---

### 5. 🔗 마인드맵 시맨틱 확장

한 키워드를 깊이 파고드는 다단계 발굴 도구.

**4중 확장**
- 자동완성 (PC + 모바일 + 쇼핑 + 연관)
- 시맨틱 sibling (헤드 명사 공유)
- 롱테일 (의도 어미 추가)
- 다단계 (1~5 깊이)

**옵션**
- 50개 (1단계) / 150 (2단계) / 300 (3단계) / 400 (4단계) / 500 (5단계)
- 각 후보의 sv/dc 실측 + 등급 자동 분류 (SSS/SS/S/A/B)
- 엑셀 일괄 내보내기

---

### 6. 📎 신호 근거 보기 (신뢰성 — 다른 도구가 못 하는 기능)

모든 홈판 카드 우하단 [📎 근거 보기] 버튼.

**출처 3단계 분류**
- 🟢 **공식**: C-Rank, D.I.A., D.I.A.+, AiRS 등 네이버 발표 (트윈워드/InterAd/SEO NEWS 등 cross-verify)
- 🟡 **업계 표준**: 블랙키위, 키워드인사이트, 황금키워드마스터 등 다중 도구 합의
- 🔴 **휴리스틱**: LEWORD 자체 (공식·표준 합의 없는 신호)

각 출처 URL과 핵심 인용문이 클릭 한 번에 확인 가능.
"왜 이 점수가 나왔는지" 사용자가 직접 검증할 수 있습니다.

## ⚠️ LEWORD가 약속하지 않는 것 (솔직한 한계)

### 1. 메인 홈피드 진입 100% 보장 불가

- 네이버 홈피드는 **AiRS 개인화 추천** 알고리즘 — 사용자별 임베딩으로 결정
- 어떤 도구도 보장 불가, LEWORD도 마찬가지
- 우리가 약속하는 건: **"검증된 신호 동시 충족 시 통계적 노출 우위"**

### 2. 시맨틱 확장은 룰 기반 (LLM 아님)

- LEWORD는 OpenAI/Claude 같은 LLM 의미 분석을 **사용하지 않습니다**
- 한국어 형태소 분석 없이 휴리스틱(헤드 명사 + 자동완성 다단계 + 토큰 매칭) 사용
- 장점: 빠르고 안정적, API 비용 사용자에 떠넘기지 않음
- 한계: 진짜 동의어("민생지원금" ↔ "긴급생계지원금" 같은) 매칭은 어려움

### 3. 노출 추적은 검색 SERP 기준만

- 메인 홈피드 추적 불가 (위 #1 사유)
- 검색 SERP 노출 자체가 "잘 만든 글" 신호로 활용됩니다
- 신생/저권위 블로그는 격전지 키워드에서 top 30 진입 어려움 — **진짜 결과 그대로** 보여드림

### 4. "AI" 라고 주장하지 않습니다

- 대부분 룰 기반 휴리스틱 + 네이버 공식 API + SERP 크롤
- 일부 기능(추천 제목 일부 변주)에 옵션으로 AI 호출 가능 — 기본은 비활성

## ❓ 자주 묻는 질문

**Q1. 다른 키워드 도구와 무엇이 다른가요?**
- 블랙키위/키워드인사이트는 발굴까지
- LEWORD는 발굴 + **시맨틱 sibling 확장** + **내 글 노출 자동 추적** + 26 카테고리 헌터 통합
- 모든 수치 실측 + **출처 명시 (다른 도구 X)**

**Q2. AI/LLM API 키 필요한가요?**
- 아니요. 핵심 기능은 모두 네이버 공식 API + SERP 크롤 기반
- 추가 API 비용 없음

**Q3. 한 PC에서만 쓰나요?**
- 라이선스 1개당 1 PC
- PC 변경 시 라이선스 재발급 가능

**Q4. 환불 정책은?**
- 결제 후 7일 이내 사용 미발생 시 100% 환불
- 상세 약관은 결제 시 확인

**Q5. macOS도 가능한가요?**
- 가능합니다. macOS dmg 별도 빌드 제공

**Q6. 자동 업데이트되나요?**
- 네. electron-updater로 새 버전 출시 시 자동 알림 + 업데이트

**Q7. 데이터는 어디에 저장되나요?**
- 모든 데이터는 사용자 PC 로컬에만 저장 (`%APPDATA%/leword`)
- 외부 서버 전송 없음 (라이선스 인증 제외)

**Q8. 메인 홈판 노출 보장하나요?**
- **보장하지 않습니다.** 네이버 홈피드는 AiRS 개인화 추천이라 본질적으로 블랙박스
- 다만 검증된 신호 동시 충족 시 통계적 노출 우위는 약속
