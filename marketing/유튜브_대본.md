## 1. 짧은 광고 (30~45초) — 핵심 메시지 하나

### 영상 1A. "네이버 연관키워드 종료" 대응

```
[0~3초] 검정 화면 + 흰 자막

자막: "네이버 연관 키워드가 사라졌습니다."
효과음: 짧은 알림 소리

[3~8초] 빈 자리 강조

자막 페이드인: "이제 어디서 키워드를 찾으시나요?"

[8~18초] LEWORD 화면 캡처 — 💎 황금 키워드 자동 발굴 결과

내레이션:
"LEWORD는 네이버 연관 키워드의 상위호환입니다.
입력한 키워드 하나로 — 헤드 명사를 공유하는 의외의 황금 키워드를
검색량·문서수·황금비율 실측까지 자동 발굴합니다."

화면: '황금키워드' 입력 → '꿀키워드 / 롱테일 키워드 발굴 / 타겟 키워드' 결과 표시

[18~28초] 차별점 자막

자막 (한 줄씩 페이드인):
"✓ 시맨틱 sibling 확장 (단순 자동완성 X)
 ✓ 네이버 검색 API 실측 (추정값 X)
 ✓ 황금비율 자동 필터 (검색량≥300, 문서수≤10000, 비율≥5)"

[28~38초] 콜투액션

내레이션: "한 달 3만 원, 3달 10만 원, 1년 30만 원.
            네이버 연관 키워드가 종료된 지금, LEWORD가 그 자리를 채웁니다."

자막: "LEWORD - 키워드마스터"
화면: 다운로드 링크 / GitHub 릴리스 URL
```

### 영상 1B. "내 글 노출 자동 추적"

```
[0~5초] 사용자 시나리오

자막: "내 블로그 글이 검색되긴 하나?"
화면: 블로그 발행 후 답답한 표정 (스톡 이미지)

[5~15초] LEWORD 노출 추적 화면

내레이션:
"내 블로그 주소만 등록하면 — 자동으로 RSS 수집,
글 제목에서 핵심 키워드 자동 추출, 네이버 검색 SERP 자동 측정.
하루 한 번 모달만 열어도 hit rate가 자동으로 누적됩니다."

화면: 📊 내 노출 추적 모달 → 자동 파이프라인 진행 배너 → 대시보드 결과

[15~25초] 결과 화면

자막 (실측 강조):
"✓ 네이버 검색 API 우선 사용 (차단 없음)
 ✓ HTTP SERP + Playwright 폴백
 ✓ 카테고리별 hit rate 자동 집계"

[25~35초] 솔직한 한계 + CTA

내레이션 (솔직 톤):
"메인 홈피드는 개인화 추천이라 추적 불가지만 — 검색 SERP 노출은
실측 그대로 보여드립니다. 결과가 적으면 적은 대로 정직하게."

자막: "LEWORD - 한 달 3만 원부터"
```

## 2. 풀 데모 (3~5분) — 실제 사용 흐름

```
[0~15초] 인트로

화면: LEWORD 로고 + "Korean Blog Keyword Master"
내레이션:
"안녕하세요. 오늘은 LEWORD가 어떻게 네이버 블로그 글의 키워드를 발굴하고
실제 노출까지 자동 추적하는지 5분 안에 보여드립니다.
* 모든 점수는 네이버 API/SERP 실측 — 추정값 아닙니다."

[15~60초] 챕터 1 — 💎 황금 키워드 자동 발굴 (메인 기능)

화면: LEWORD 메인 → 키워드 입력란 "황금키워드" 입력 → [💎 황금 키워드 자동 발굴] 클릭

내레이션:
"기존 '키워드 조회'가 자동완성/연관검색만 쓴다면,
황금 발굴은 시맨틱 sibling — 헤드 명사가 같지만 modifier가 다른 키워드를 찾습니다.
예를 들어 '황금키워드' 입력 → '꿀키워드', '롱테일 키워드 발굴', '타겟 키워드'
같이 의외의 동의어/유사어가 나오죠."

화면: 진행 로그 → 결과 테이블 (검색량/문서수/황금비율)

내레이션:
"각 후보의 검색량·문서수를 네이버 SearchAd API로 실측 →
검색량 ≥300, 문서수 ≤10000, 황금비율 ≥5인 것만 자동 필터.
가짜 추정값이 아닌, 진짜 발굴된 가치 키워드만 보입니다."

[60~135초] 챕터 2 — 🏠 홈판 키워드 헌터 (카테고리 기반)

화면: 추가 기능 → 🏠 홈판 키워드 헌터 모달 → 26 카테고리 중 선택 → 발굴

내레이션:
"카테고리를 선택하면 — beauty/fashion/food/...
26개 카테고리에 대해 LEWORD가 미리 정한 시드 + 자동완성으로
키워드를 발굴하고, 4개 실측 신호(CTR가능성·신선도·카테고리·빈자리)로
homeScore를 산출합니다."

화면: 결과 카드 → [📎 근거 보기] 클릭 → 출처 모달

내레이션:
"각 신호의 출처는 명시 — C-Rank/D.I.A.+ 같은 네이버 공식,
블랙키위/키워드마스터 같은 업계 표준, LEWORD 자체 휴리스틱으로
3단계 분류. 메인 홈피드는 블랙박스라 보장은 못 하지만,
검증된 신호 동시 충족 시 노출 우위는 통계적으로 약속드립니다."

[135~210초] 챕터 3 — 📊 내 노출 추적 (검증 루프)

화면: 추가 기능 → 📊 내 노출 추적 → RSS URL 입력

내레이션:
"네이버 블로그 ID나 주소만 입력하면 — 자동으로 RSS URL 변환,
글 50건 자동 수집, 글 제목에서 핵심 키워드 자동 추출,
페어 자동 등록까지. 사용자는 하나도 손 안 댑니다."

화면: 자동 파이프라인 배너 진행 → SERP 체크 → 대시보드

내레이션:
"체크는 네이버 블로그 검색 API 우선, 차단 시 Playwright 폴백.
각 페어는 measured 상태 → 카테고리별 hit rate 자동 집계.
2~3주 누적되면 본인 블로그의 강점/약점 카테고리가 드러납니다."

화면: [➕ 특정 글 노출 확인] 버튼 클릭 시연

내레이션:
"자동 매칭이 못 잡은 글은 키워드 + 글 URL 직접 입력해서 측정 가능.
다른 블로그로 바꿀 땐 [🗑️ 모든 데이터 초기화] 한 번이면 깔끔."

[210~255초] 챕터 4 — 🆕 연예 갓 떴음 (선점 가능 기사)

화면: 메인 페이지 → 연예 갓 떴음 카드 (4매체 통합) — 분 단위 시간 + 🟢 선점 / ⚠️ 경쟁

내레이션:
"이미 HOT인 기사는 글쓰기엔 늦었죠. LEWORD는 스타뉴스 +
스포츠조선 + 디스패치 + 마이데일리 4개 매체에서
지금 막 떴는데 아직 HOT 진입 안 한 — 선점 가능 기사만 자동 필터해
한 화면에 보여드립니다. 5분마다 자동 갱신."

[255~290초] 챕터 5 — 마인드맵 시맨틱 확장

화면: 키워드 입력 → [🔗 마인드맵] → 결과 트리

내레이션:
"키워드 하나에 대해 — 자동완성 + 연관검색 + 시맨틱 sibling 4중 확장.
50~500개 후보를 다단계로 발굴하고, 등급(SSS/SS/S/A/B)으로 자동 분류.
엑셀로 한 번에 내보내기도 됩니다."

[290~310초] 챕터 6 — 트레이 아이콘화 (UX)

화면: 우상단 [⬇️ 트레이로] 버튼 클릭 → 창 사라짐 → 트레이 아이콘

내레이션:
"발굴 결과를 보존한 채 트레이로 최소화 — 작업 중에 방해 없이.
[X = 트레이] 체크하면 X 버튼도 트레이행."

[310~340초] 아웃트로 + 솔직한 한계

내레이션:
"LEWORD는 '메인 홈판 100% 보장'을 약속하지 않습니다.
네이버 홈피드는 AiRS 개인화 추천이라 어떤 도구도 보장 불가.
다만 — C-Rank·D.I.A.+ 공식 신호 + 업계 표준 + 자체 휴리스틱을
한 화면에서, 모든 수치를 실측으로, 출처까지 명시해 제공합니다.

한 달 3만 원, 3달 10만 원, 1년 30만 원.
네이버 연관 키워드가 사라진 지금, LEWORD가 그 자리를 채웁니다."

화면: LEWORD 로고 + 다운로드 링크
```

## 가이드라인 (이 대본을 영상화할 때)

**필수**
- 모든 수치 클레임에 "실측" / "네이버 API" / "SERP 크롤" 명시
- "보장" 단어 사용 금지 — "통계적 우위" / "확률 상승" 만 허용
- 베타 테스터 후기, 가짜 사용자 인터뷰 금지

**권장**
- 화면 캡처는 실제 LEWORD 실행 화면 (조작 가능하지만 합성 X)
- 솔직한 한계 (메인 홈피드 추적 불가, 격전지에서 결과 부족 등) 도 1줄씩 명시
- 가격은 영상 중간에 1회 + 끝에 1회만

**금지**
- "X배 노출" 같은 정량 약속 (백테스트 데이터 없음)
- 경쟁 도구(블랙키위/키워드인사이트) 직접 비교
- LEWORD가 "AI" 라고 주장 (대부분 룰 기반 — 표현 정직하게)
