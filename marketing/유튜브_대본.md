## 영상 4 · 시리즈 짧은 컷 (각 30~45초)

### 시리즈 A — "이게 왜 다른가" (3편)

**A1. "시맨틱 sibling" 시연**
- 키보드로 "황금키워드" 입력
- 다른 도구 결과: "황금키워드 찾는법", "황금키워드 추천" (prefix만)
- LEWORD 결과: "꿀키워드", "롱테일 키워드 발굴", "타겟 키워드" (의외의 sibling)
- 차이 시각화 → "이게 시맨틱 sibling 입니다"

**A2. "신호 근거 보기"**
- 홈판 결과 카드 → [📎 근거 보기] 클릭
- 모달에 [공식] [업계 표준] [휴리스틱] 3가지 라벨
- 원문 URL + 인용문 표시
- "다른 도구는 점수만, LEWORD는 출처까지"

**A3. "노출 자동 추적"**
- RSS 등록 → 50건 자동 수집 → SERP hit rate
- "글 발행하고 끝이 아니라, 정말 노출됐는지 자동 검증"

### 시리즈 B — "Q&A 형식" (3편)

**B1. "메인 홈판 보장하나요?" → "보장 안 합니다 — 솔직한 이유"**
- 사용자 질문 자막 → LEWORD 솔직 답변
- "AiRS 알고리즘은 블랙박스, 우리도 마찬가지"
- "대신 검증된 신호로 통계적 우위는 약속"

**B2. "AI 쓰나요?" → "거의 안 씁니다"**
- "대부분 룰 기반 휴리스틱"
- "AI API 비용 사용자에 떠넘기지 않습니다"
- "느린 LLM 호출 대신 빠른 결정론적 측정"

**B3. "다른 도구와 차이?" → "발굴 + 추적 통합"**
- 블랙키위/키워드인사이트: 발굴까지
- LEWORD: 발굴 + 시맨틱 sibling + 노출 자동 추적

### 화면 캡처 권장 시나리오

1. **클린 인스톨된 LEWORD 사용** (테스트 데이터 없음)
2. **실제 키워드 입력** ("황금키워드", "재테크", "다이어트" 등 일반)
3. **결과는 실측 그대로 보여주기** — 0건이어도 OK (솔직)
4. **카메라/마이크는 깔끔하게** — 배경음 ↓, 내레이션 명확
